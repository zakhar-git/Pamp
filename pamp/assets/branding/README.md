Pamp report branding assets
===========================

Logo supports png, jpg, jpeg, webp, gif, mp4, webm.
Background supports png, jpg, jpeg, webp.

Recommended logo size: 512x512.
Recommended background size: 1920x1080.

Pamp chooses the first existing logo in this order:
logo.mp4, logo.webm, logo.gif, logo.webp, logo.png, logo.jpg, logo.jpeg.

Pamp chooses the first existing background in this order:
background.webp, background.png, background.jpg, background.jpeg.
