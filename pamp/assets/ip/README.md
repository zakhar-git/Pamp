# IP report assets

- `world/world.svg`: `@svg-maps/world` 2.0.0, MIT license. Country paths use ISO 3166-1 alpha-2 IDs.
- `flags/*.svg`: `flag-icons` 7.5.0, MIT license. Files use lower-case ISO 3166-1 alpha-2 names.

The HTML exporter embeds the selected flag and the world map into the generated report so IP reports remain fully functional over `file://`.
